# Guía 2: Consolidación en C++

En esta sección se encuentran los ejercicios correspondientes a la segunda parte de la materia, enfocados en la aplicación práctica de tipos de datos y lógica inicial.

## Contenidos:

* Manejo de flujos de datos y facturación simple.
* Estructuras para el manejo de información de usuarios y estudiantes.
* Entrada de datos complejos (mascotas y empleados).
* Registro y organización de datos de vehículos.
* Uso de operadores aritméticos para cálculos de importes.
