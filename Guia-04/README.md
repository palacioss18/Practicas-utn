# Guía 4: Estructuras de Decisión en C++ (parte 2)

En esta sección se encuentran los ejercicios correspondientes a la cuarta guía, enfocados en el uso de estructuras condicionales y la toma de decisiones en programas simples.

## Contenidos:

* Aplicación de estructuras condicionales (`if`, `else`).
* Uso de `switch` para selección de opciones.
* Comparación de valores numéricos.
* Validación de datos ingresados por el usuario.
* Clasificación de información (edad, estados, resultados).
* Uso de ciclos (`while`,`for`) para repetir instrucciones.
* Implementación de cálculos básicos (suma,resta,multiplicaciones).

## Ejercicios incluidos:

* Programa para comprobar si un numero es par o impar.
* Contador de despegue.
* Contar numeros positivos e negativos.
* Suma acumulada en un while.
* Suma hasta que el usuario ingrese 0.
* Pedí un número N y sumá todos los números desde 1 hasta N.
