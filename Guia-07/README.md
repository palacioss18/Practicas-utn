# 📘 Guía 07 - Práctica con `for` en C++

Este repositorio contiene ejercicios realizados para practicar estructuras repetitivas (`for`) en C++.

## 📂 Contenido

* `guia7-ejercicio1.cpp` → Conteo inverso
* `guia7-ejercicio2.cpp` → Contar del 1 al 5
* `guia7-ejercicio3.cpp` → Números pares del 1 al 20
* `guia7-ejercicio4.cpp` → Suma hasta un número dado
* `guia7-ejercicio5.cpp` → Tabla de multiplicar

## 🧠 Objetivo

Practicar:

* Uso de bucles `for`
* Variables acumuladoras
* Condicionales dentro de ciclos

## ⚙️ Cómo ejecutar

1. Compilar:

```
g++ nombre_del_archivo.cpp -o programa
```

2. Ejecutar:

```
./programa
```
